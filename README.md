# File Replicator

## 📌 Project Overview
**File Replicator** is a planned utility designed to synchronize or replicate files from a specified source folder to a destination folder. 

### 🎯 Objective
The primary goal of this tool is to provide a reliable way to mirror files between directories, ensuring that the destination folder is an accurate replica of the source.
